Release v0.43.6 - Исправление вставки текста и стабильности

v0.43.6:
- Исправлена двойная вставка текста при использовании Ctrl+V
- Добавлена поддержка Ctrl+V на любой раскладке клавиатуры
- Добавлено контекстное меню для поля ввода (ПКМ → Вставить/Копировать/Вырезать)
- Убран лишний вызов paste_text через lambda

v0.43.5:
- Исправлено смешивание истории личных сообщений с общим чатом
- Добавлено сохранение всех личных чатов в private_chats.json
- Личные чаты теперь отображаются постоянно, а не только после нажатия ПКМ
- Добавлена подсветка текущего чата в списке
- История личных сообщений сохраняется локально в отдельных файлах
- Кнопка «История» показывает только историю общего чата
- При получении личного сообщения чат автоматически добавляется в список

v0.43.4:
- Исправлено зависание клиента при загрузке истории
- Убраны проблемные lambda в цикле отображения сообщений
- Исправлено отображение личных сообщений (правильное определение отправителя)
- Добавлена задержка при закрытии после кика/бана

v0.43.3:
- Добавлено отображение версии клиента в заголовке окна
- Исправлено дублирование истории чата при загрузке
- Цвета ников теперь корректно сохраняются и загружаются
- История личных сообщений хранится на сервере в private_messages.json
- При открытии личного чата клиент запрашивает историю с сервера

v0.43.2:
- Файлы теперь привязаны к чатам (общие файлы видны только в общем чате)
- Добавлены кликабельные ссылки в сообщениях
- Добавлена кнопка «Забыли пароль?» с генерацией кода восстановления
- Код восстановления выводится в консоли сервера

v0.43.1:
- Интерфейс переработан в стиле Telegram
- Добавлена левая панель со списком чатов (общий + личные)
- Личные сообщения открываются через ПКМ по пользователю
- Добавлена RGB-палитра для выбора цвета ника
- Админ-функции (мут/кик/бан) убраны из клиента, только через консоль сервера
- Файлы теперь отображаются в правой панели
- Добавлена кнопка обновления списка файлов
