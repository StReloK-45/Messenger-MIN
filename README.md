## 🚧 О проекте

**MIN** — это упрощённый локальный чат для работы в офисе и классах информатики для обмена данными между коллегами или студентами. Он позволяет быстрое создание сервера через хост-машину, и простой запуска клиента для пользователей. Имеет базовый достаточный функционал для комфортного общения между участниками чата.

---

## ✨ Ключевые особенности

| Особенность | Описание |
|---------|----------------|
| **Файловая система** | Очень удобная файловая система. Загружаешь на сервер файл, и он появляется в каталоге для скачивания другими участниками чата. |
| **Личные сообщения** | Свободная система личных сообщений, в котором каждый участник может общаться лично с другим участником. Все диалоги, которые вы когда-либо начинали, сохраняются в левой панели и не пропадают после перезапуска. |
| **История чата** | История личных переписок сохраняется как на сервере, так и локально у клиента. |
| **Система авторизации** | Удобная и быстрая система регистрации и авторизации аккаунтов в чаты. |
| **Цвет ника** | У каждого пользователя свой уникальный цвет. Цвет можно изменить в любой момент через встроенную RGB-палитру для разнообразия и индивидуальности в переписке. |
| **Контекстные меню** | Меню по правому клику в области чата и в поле ввода: Копировать/Вставить/Вырезать |
